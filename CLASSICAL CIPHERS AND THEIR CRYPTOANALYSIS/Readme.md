- `CaesarCipher` - зашифровать текст с помощью шифра Цезаря
- `PlayferCipher` - дешифровать текст с помощью шифра Плейфера
- `RailFenceCipher` - дешифровать текст с помощью шифра частоколу
- `HillsCryptographicSystem` - дешифровать текст с помощью криптосистемы Хилла
- `VisenersCipher` - зашифровать текст с помощью шифра Виженера
- `FrequencyAnalysisCaesarCipher` - с помощью частотного криптоанализа расшифровать текст шифра Цезаря
- `KaziskiFriedmanMethod` - криптоанализ и дешифровка шифра Виженера с помощью Метода Казиски и Метода Фридмана. (+ использовался критерий метрики Пирсона х^2)
